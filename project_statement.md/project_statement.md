## Project Statement

The project titled **“Fake Product Review Detection using Text Classification (Basic NLP)”** focuses on addressing the growing issue of fraudulent reviews in e-commerce platforms. Fake reviews can mislead customers, artificially inflate product ratings, and damage the credibility of online marketplaces.

The system leverages text preprocessing, Bag of Words feature extraction, and a Logistic Regression classifier to analyze review content.  

The workflow includes:  
- Loading a dataset of product reviews with labels (1 = Fake, 0 = Genuine).  
- Preprocessing and splitting the data into training and testing sets.  
- Converting textual reviews into numerical features using **CountVectorizer**.  
- Training a **Logistic Regression** model to distinguish between fake and genuine reviews.  
- Evaluating the model using accuracy and classification metrics.  
- Predicting the authenticity of new, unseen reviews.  

By automating the detection of fake reviews, this project contributes to enhancing trust and transparency in online shopping, helping consumers make informed decisions and supporting e-commerce platforms in maintaining credibility.
